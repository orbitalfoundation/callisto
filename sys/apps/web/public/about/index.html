Orbital About



Orbital is an app framework for creating rich apps with an emphasis on self sovereign identity, granular security, shared state, persistence, distributed computation and 3d user interfaces.

It introduces the idea of a 'scenario definition language' where applications are wired together out of dynamically loaded services such as computer vision modules, deep learning modules and rendering and visualization modules.

These apps currently run within a web container but the intent is to also run natively on dedicated hardware such as next generation augmented reality glasses.

Orbital is licensed under an Apache/MIT open source license and our source code is available on github.com/orbitalweb. Our ongoing thinking and development is captured on here orbitalweb.github.io
